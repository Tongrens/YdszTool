## 使用方法
本工具适用于深职院韵动深职公众号的场地预约，按要求填写信息即可

若勾选“定时提交预约”则为定时预约，否则为单次预约，不勾选则预约按钮上的时间无效

**本程序不会保存并上传任何的个人信息，请放心使用**
